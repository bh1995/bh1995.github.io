Projects


  • Heart disease classification using transformers in Pytorch:

    Before Generative LLMs became so hyped in main stream media, they were hyped within the machine learning community. The OG "Attention is All You Need" paper showcased the power of the transformer architecture. In this project I showcase how this classic language model can be re-constructed to perform classification on any sequential data input. More specifically I use real ECG signals to train a model that achieves high accuracy in predicting a common heart disease called Atrial fibrillation.

    Medium blog post, Github

  • Cycle GAN in Pytorch:

    The earliest implementation of generative AI (as far as I know) was created by Ian Goodfellow using a GAN architecture. Deriving from the orignal GAN idea, the highly successful Cycle GAN was created. In this project I implement the Cycle GAN which works by converting zebras to horses and vice-versa. See the medium blog post link for an in-depth description of the project and the Github link for the project code.

    Medium blog post.
